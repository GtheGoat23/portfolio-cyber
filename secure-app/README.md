# Application Sécurisée – Guide

## Objectif
Concevoir une petite API (Flask/Node) avec contrôles de sécurité:
- Authentification (JWT)
- Validation d'entrées
- Journalisation (logs) et rate limiting
- Tests (unitaires + ZAP)

## Délivrables
- Code + README
- Rapport de menaces (threat modeling)
- Résultats SAST/DAST
